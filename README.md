#  AudioRedactor

**Простой аудиоредактор для обработки WAV-файлов**


##  Описание проекта

**AudioRedactor** — это настольное приложение с графическим интерфейсом для базовой обработки аудиофайлов формата WAV.  Программа разработана на языке Python с использованием стандартной библиотеки Tkinter и модуля wave и **не требует установки сторонних зависимостей** для работы с аудио.

### Ключевые особенности:
-  Полностью автономная работа без внешних библиотек для обработки аудио
-  Интуитивно понятный графический интерфейс
-  Поддержка WAV-файлов различной битности (8, 16, 24, 32 бит)
-  Работа с моно и стерео аудио
-  Покрытие кода модульными тестами

---

##  Функциональные возможности

| Функция | Описание |
|---------|----------|
| **Загрузка аудио** | Открытие WAV-файлов с отображением метаданных (длительность, каналы, частота дискретизации) |
| **Обрезка аудио** | Выделение фрагмента аудио по заданным временным меткам (начало и конец в секундах) |
| **Изменение громкости** | Регулировка громкости в диапазоне от -50 до +50 дБ с предупреждением о возможных искажениях |
| **Сохранение результата** | Экспорт обработанного аудио в WAV-формат |

##  Установка и запуск

### Системные требования

- Python 3. 6 или выше
- Операционная система: Windows / macOS / Linux

### Установка

1. **Клонируйте репозиторий:**

```bash
git clone https://github.com/IlyaRassylshchikov/AudioRedactor. git
cd AudioRedactor
```

2. **Запустите приложение:**

```bash
python Audio_redactor_Rassylshikov. py
```

>  **Примечание:** Дополнительные библиотеки не требуются!  Приложение использует только стандартные модули Python: `tkinter`, `wave`, `struct`, `math`, `os`.

##  Руководство пользователя

### Шаг 1: Загрузка аудиофайла

1. Нажмите кнопку **"Выбрать файл (WAV)"**
2. В диалоговом окне выберите WAV-файл
3. После загрузки отобразится информация о файле:
   - Длительность (в секундах)
   - Количество каналов (1 — моно, 2 — стерео)
   - Частота дискретизации (в Гц)

### Шаг 2: Обрезка аудио

1.  Введите время начала фрагмента (в секундах)
2. Введите время конца фрагмента (в секундах)
3.  Нажмите кнопку **"Обрезать"**

### Шаг 3: Изменение громкости

1. Установите значение изменения громкости в децибелах:
   - Положительные значения (+dB) — увеличение громкости
   - Отрицательные значения (-dB) — уменьшение громкости
2. Нажмите кнопку **"Применить"**

>  При изменении громкости более чем на ±50 дБ появится предупреждение о возможных искажениях. 

### Шаг 4: Сохранение результата

1.  Нажмите кнопку **"Сохранить как..."**
2.  Выберите расположение и имя файла
3. Файл будет сохранён в формате WAV

## Техническая реализация

### Обработка аудио (Audio_processor_Rassylshikov.py)

Класс `AudioProcessor` реализует низкоуровневую работу с WAV-файлами:

- **Загрузка файла:** Использует модуль `wave` для чтения параметров и фреймов аудио
- **Обрезка:** Вычисляет позиции байтов на основе временных меток и частоты дискретизации
- **Изменение громкости:** Применяет формулу преобразования децибел в линейный коэффициент:
  
  ```
  factor = 10^(dB / 20)
  ```
  
  Каждый сэмпл умножается на коэффициент с ограничением по максимальному значению для предотвращения клиппинга.

### Графический интерфейс (Audio_redactor_Rassylshikov.py)

- Построен на основе `tkinter` с использованием `LabelFrame` для логической группировки элементов
- Реализована обработка ошибок с информативными сообщениями
- Статусная строка отображает текущее состояние операций
